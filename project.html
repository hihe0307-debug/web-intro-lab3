Internet Website Browser Sever

Internet là gì?

Internet là hệ thống mạng máy tính toàn cầu giúp các thiết bị kết nối và trao đôi dữ liệu với nhau.

Web là gì?

Web là tập hợp các trang web được liên kết với nhau thông qua Internet.

Website và trang Web

Trang web là một trang riêng rẻ trong website.

Browser (Trình duyệt)

Browser là phần mềm dùng để truy cập website.

Sever là gì?

Sever là máy chủ dùng để lưu trữ dữ liệu website.
