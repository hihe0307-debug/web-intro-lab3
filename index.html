Phân biệt Internet và Web

Internet là mạng lưới phần cứng toàn cầu kết nối các máy tính lại với nhau.

Web (World Wide Wed) là một dịch vụ chạy trên Internet, chứa các trang web kết nối bằng liên kết.

Các khái niệm cơ bản:

  • HTML: Ngôn ngữ tạo cấu trúc trang web.
  • HTTP: Giao thức truyền tải dữ liệu web.
  • Trình duyệt: Công cụ để xen trang web (Chrome, Safari,...).
